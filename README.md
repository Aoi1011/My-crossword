## Finite Fields

### Fermat's little theorem
n ^ (p-1) is always 1 for every p that is prime and every n > 0.

n ^ (p-1) % p = 1 where p is prime


Division in finite fields
a/b = a*(1/b) = a*b^-1

b^-1 = b^-1*1 = b^-1*b^(p-1) = b^(p-2)

p = 19

2/7 = 2*7^(19-2) = 2*7^17 % 19 = 3
